# 🤖 AI Recruitment Agent

An intelligent resume screening and candidate matching system built using **Python, Streamlit, and NLP**. This app helps recruiters automatically analyze resumes and find the best candidates based on job descriptions.

---

## 🚀 Features

* 📄 Upload and analyze resumes (PDF)
* 🧠 NLP-based text processing using spaCy
* 📊 Similarity scoring using TF-IDF & Cosine Similarity
* 📈 Visual representation of candidate scores (Pie Chart)
* ⚡ Fast and interactive UI using Streamlit
* 🧩 Easy-to-use interface for recruiters

---

## 🛠️ Tech Stack

* **Frontend & App UI:** Streamlit
* **Backend Logic:** Python
* **NLP:** spaCy
* **Data Processing:** Pandas
* **Machine Learning:** Scikit-learn
* **Visualization:** Matplotlib

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/shruti-coon/AI-Recruitment-Agent.git
cd AI-Recruitment-Agent
```

### 2. Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the app

```bash
streamlit run app.py
```
